The QosClass Enum

(PHP 8 >= 8.4.0)

Introduction

The QosClass enum is used to specify the user process priority with pcntl_setqos_class().

Enum synopsis

enum QosClass
{
case Pcntl::Background ; // Runs the process after all high-priority ones had ran their courses.

case Pcntl::Default ; // Runs the process after all high-priority processes but before the low-priority ones.

case Pcntl::UserInteractive ; // Runs the process as the highest priority level.

case Pcntl::UserInitiated ; // Runs the process at high priority level but below Pcntl::UserInteractive ones.

}
